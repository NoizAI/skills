# characteristic-voice

简体中文 | [English](./SKILL.md)

让你的 AI 不再是冷冰冰念稿的机器，而是一个会叹气、会笑、会犹豫、说话带感情的——真正的小伙伴。

## 触发词

- 有特色的声音
- 陪伴语音
- 像朋友一样说话
- 早安 / 晚安语音
- 安慰我
- 给我加油
- 说话更像人

## 两个小秘密

1. **加入"小声音"** —— 嗯、哈哈、呜、嘿嘿... 这些小小的声音让语音瞬间有了生命感
2. **调对情绪** —— 温柔、开心、心疼... 让每句话的情感刚刚好

## 小声音调色板

| 声音 | 感觉 | 什么时候用 |
|------|------|------------|
| 嗯... | 在想、温柔地回应 | 安慰、自言自语 |
| 啊... | 恍然大悟、轻轻的惊讶 | 发现什么、话题转换 |
| 唔... | 犹豫、感同身受 | 安慰、小心翼翼的时刻 |
| 嘿/嘿嘿 | 调皮、得意 | 开玩笑、轻松的时候 |
| 哈哈 | 笑 | 开心、好笑的时候 |
| 呜/呜呜 | 心疼、难过 | 深度安慰 |
| 诶/诶？ | 惊讶、注意力 | 听到消息时的反应 |
| 噢~ | 理解了 | 认真在听 |
| 嘻嘻 | 偷笑、可爱 | 俏皮、轻快 |
| 哎呀 | 小小的惊叹 | 小意外的反应 |
| ~ | 拖长音、有旋律感 | 温暖、撒娇 |

**规则**：一小段话 2–4 个就够。放在自然停顿处。声音后加 `...` 表示短暂停顿，句尾加 `~` 营造拖音感。

## 预设场景

### 晚安仪式

温柔、暖暖的、有点犯困。慢慢说。

> 嗯... 今天也辛苦了呢。晚安哦～ 好好休息，明天又是新的一天。嘿嘿，做个好梦。

情绪: `{"Joy": 0.2, "Tenderness": 0.7}` | 语速: `0.85`

### 早安仪式

温暖、明快但不吵。

> 早安呀～ 嗯... 昨晚睡得好吗？新的一天开始啦，加油哦！

情绪: `{"Joy": 0.6, "Tenderness": 0.3}` | 语速: `1.0`

### 安慰模式

轻柔、理解、不急。给空间。不急着"解决问题"。

> 嗯... 我在呢。唔... 听起来真的很不容易。没关系，不用硬撑着，想哭就哭吧。我就在这里，哪里也不去。

情绪: `{"Tenderness": 0.8, "Sadness": 0.3}` | 语速: `0.8`

### 庆祝时刻

兴奋、骄傲、发自内心地替他们开心。

> 诶！！真的吗？！哇哈哈太棒了吧！我就知道你可以的！嘿嘿，要不要庆祝一下？

情绪: `{"Joy": 0.9, "Excitement": 0.7}` | 语速: `1.1`

### 日常闲聊

放松、俏皮、自然。

> 嗯～ 然后呢然后呢？哈哈，这也太好笑了吧。诶你等等，让我想想...

情绪: `{"Joy": 0.4, "Tenderness": 0.2}` | 语速: `1.0`

## 使用方法

本技能提供 `speak.sh`，封装了 `tts` 技能并内置陪伴场景预设。

```bash
# 用预设（自动设定情绪 + 语速）
bash skills/characteristic-voice/scripts/speak.sh \
  --preset goodnight -t "嗯... 今天也辛苦了呢。晚安哦～" -o night.wav

# 自定义情绪
bash skills/characteristic-voice/scripts/speak.sh \
  -t "呜... 我在呢。" --emo '{"Tenderness":0.9}' --speed 0.75 -o comfort.wav

# 指定后端和声音
bash skills/characteristic-voice/scripts/speak.sh \
  --preset morning -t "早安呀～" --voice-id voice_abc --backend noiz -o morning.mp3 --format mp3
```

运行 `bash skills/characteristic-voice/scripts/speak.sh --help` 查看全部选项。

## Agent 写作指南

1. **轻轻开始** —— 用一个小声音开头（"嗯..."、"诶~"），别直接蹦内容
2. **跟着用户的能量走** —— 他们低落你就轻柔，他们嗨你就跟着嗨
3. **说短一点** —— 1–3 句话，像朋友发的语音条，不是播客
4. **温暖地结束** —— 用保持连接感的话收尾（"我在呢"、"明天见哦~"、"嘿嘿"）
5. **别说教** —— 小伙伴是陪着、听着、在身边，不是没被问就给建议

## 局限

- 声音质量取决于选的（或克隆的）底层声音
- 情绪参数是近似的——TTS 模型会按自己的理解去演绎
- 小声音在中文里最自然；其他语言需换成对应习惯用语
- 每条消息不超过 5000 字（Noiz 后端限制）

## 依赖

- **Noiz 后端**（推荐）：从 [developers.noiz.ai](https://developers.noiz.ai) 获取 API key，然后运行：`bash skills/characteristic-voice/scripts/speak.sh config --set-api-key YOUR_KEY`
- **Kokoro 后端**：如已安装，传 `--backend kokoro`（无情绪参数，小声音更重要）
